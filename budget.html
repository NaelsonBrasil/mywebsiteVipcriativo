Como funciona orçamento

Orçamento incia pela coleta de informações, exemplo: oque você precisa, como você quer, como sera costumização, isso sera útil em que podemos melhorar entre outros.

Como funciona os preços

Infelizmente ainda não existe uma tabela fixa. Utilizamos algumas técnicas, se for pagina web simples, cobramos por paginas concluidas de 100 a 200 reais, caso seja sistema precisa ser analizado complexbilidade para chegar em um preço para para ambos os lados.
